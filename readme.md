# DevClub – Interface de Chat

Description
Interface de messagerie instantanée moderne inspirée de WhatsApp et Discord.
"JOKKO" signifie "contacter quelqu'un" en wolof.

## Fonctionnalités

- Sidebar de contacts avec avatars
- Fenêtre de messages avec bulles (droite/gauche)
- Zone de saisie de message
- Design responsive et moderne

## Technologies utilisées

- HTML5
- CSS3 (Flexbox, Border-radius complexes, Overflow)
